# Zion-Surface

Note: Currently, Zion-Surface is PoC project. it doesn't use actually ちゃアニ backend API. We need to replace legacy it.

## Build Setup
``` bash
# run dev server on http://localhost:2015
$ cp .env.default .env
$ docker-compose up
```

## Feature
- an alternative web client for ちゃアニ
    - as PWA
    - with SSR
- automatic HTTPS
